# Restaurant Reservation System

Sistem manajemen reservasi restoran modern dengan Laravel 12.

## 🚀 Quick Start

```bash
git clone https://github.com/dwipariandi7-design/restaurant-reservation-system.git
cd restaurant-reservation-system
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan storage:link
php artisan serve
```

## 👥 Default Users
- Admin: `admin@restaurant.local` / `password`
- CS: `cs@restaurant.local` / `password`
- Customer: `customer@restaurant.local` / `password`

## 📋 Features
- ✅ Role-Based Access Control
- ✅ Sistem Reservasi Meja
- ✅ Sistem Pemesanan Menu
- ✅ Dashboard Analytics
- ✅ Laporan PDF & Excel
- ✅ Real-time Notifications
